Поймай лид — мини-игра для домашнего задания по Cursor

Как запустить:
1. Откройте файл index.html в браузере двойным кликом.
2. Нажмите «Начать игру».
3. Управление: стрелки / A-D / мышь / палец на телефоне. Пробел — AI-фильтр, когда шкала заполнена.
4. Для сдачи домашнего задания можно отправить скриншот экрана результата или загрузить проект на хостинг и отправить ссылку.

Тема игры:
Игрок управляет CRM-воронкой, ловит хорошие заявки и избегает мусорных лидов.
